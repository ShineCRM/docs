ShineCRM has four built-in AI features. They all run on Anthropic Claude models. Three of them run automatically on customer activity. One is an interactive chat you open from the menu.

## What the AI does

    Operator-facing chat for business questions, message drafting, and sending review requests.
  </Card>
  <Card title="Incoming message suggestions" icon="envelope-open-text" href="/guides/ai/incoming-message-suggestions">

    After a job completes, drafts personalized review request messages for each platform.

## What the AI does not do

The AI never sends a message, changes a pipeline stage, schedules a job, or modifies your CRM without your approval. Every automatic feature creates a **suggestion** that lands in your Command Centre. You review and approve or reject it.

The two exceptions are:

1. **Review request drafting after a job.** Drafts land in your scheduled messages list as pending follow-ups, not approved. You still review them before send time.
2. **Chat-triggered review requests.** When you explicitly tell the AI Assistant to send a review request to a specific customer, it schedules an approved follow-up one minute out so the next send cycle picks it up.

The AI does not have access to:

- Photos or attachments
- Phone call recordings
- Other organizations' data (every query is scoped to your organization)
- Payment card details

## Which models are used

| Feature | Default model |
| --- | --- |
| AI Assistant chat | Claude Sonnet 4.6 |
| Incoming message suggestions | Claude Haiku 4.5 |
| Review request drafting | Claude Haiku 4.5 |
| Message rewriter | Claude Haiku 4.5 |

Heavier reasoning (Sonnet) is used for free-form chat where quality matters most. Lighter, faster models (Haiku) are used for high-volume background tasks. Model choice is managed by ShineCRM and can be tuned per organization on request.

## Cost and usage

Every AI call is logged with token counts and a USD cost. You can see total AI usage for the current month under **Settings → Usage** (owner and admin only).

## Privacy

Your organization's data only ever flows to one external destination during AI work: the Anthropic API, for the duration of a single request. Anthropic does not train on API traffic. No data is shared with any other AI provider.
